The following commands, or symbolic links to commands, are required in /bin:
cat
chgrp
chmod
chown
cp
date
dd
df
dmesg
echo
false
hostname
kill
ln
login
ls
mkdir
mknod
more
mount
mv
ps
pwd
rm
rmdir
sed
sh
stty
su
sync
true
umount
uname

If /bin/sh is not the POSIX compatible shell command itself, it must be a hard or symbolic link to the
real shell command.

csh
ed
tar
cpio
gzip
gunzip
zcat
netstat

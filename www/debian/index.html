m68k-atari-mint cross-tools

Debian-based Linux distributions

Add the following line to your /etc/apt/sources.list, or to a new file in /etc/apt/sources.list.d. According to your distribution, replace '<mydist>' with 'xenial', 'bionic', 'eoan', 'focal', 'jammy', 'kinetic', 'bullseye', 'buster', 'stretch', or 'jessie': 

deb [signed-by=CC2E22F1B6492FD3EACFF8BF3C9569A568969A28] https://tho-otto.de/download/debian/ubuntu <mydist> main

The public key for apt-secure can be downloaded

  • here for Debian/Ubuntu distributions.

You can add this key with 

sudo apt-key add pgp-key.public

or combine downloading and registering: 

wget -q https://tho-otto.de/download/debian/ubuntu/pgp-key.public -O- | sudo apt-key add -

The key fingerprint for the public key is 

CC2E 22F1 B649 2FD3 EACF  F8BF 3C95 69A5 6896 9A28
Thorsten Otto (m68k atari mint signing key) <admin@tho-otto.de>

Update apt, and install any packages you need: 

sudo apt-get update 
sudo apt-get install cross-mint-essential
The above example will install a meta-package that installs all needed components.

You may install any additional libaries. You can get a list by using 

aptitude search cross-mint

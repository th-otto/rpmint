m68k-atari-mint cross-tools

Cygwin distribution

We provide a setup repository for cygwin. All packages are signed. The public key for cygwin can be downloaded here. You can download this key with 

wget -q https://tho-otto.de/download/cygwin/atari-mint-cross.pub

The key fingerprint is 

CC2E 22F1 B649 2FD3 EACF  F8BF 3C95 69A5 6896 9A28
Thorsten Otto (m68k atari mint signing key) <admin@tho-otto.de>

After importing the public key, setup can be invoked (from the commandline) as follows: 

setup-x86_64 --site http://tho-otto.de/download/cygwin -K atari-mint-cross.pub

(Sorry, but i could not figure out yet how to permanently add the site to the list of mirrors (it seems to be overwritten everytime setup downloads a new mirrors.lst from cygwin), or how to permanently store the key. But you can create a shortcut to setup.exe, and add the options there)
